## Core Namespaces

### CPSV-AP (`cpsv:`)

**Namespace URI:** `http://purl.org/vocab/cpsv#`  
**Standard:** Core Public Service Vocabulary Application Profile (CPSV-AP) 3.2.0  
**Specification:** https://semiceu.github.io/CPSV-AP/

**Purpose:** Core classes and properties for describing public services in the European Union.

#### Classes

| Class                | Description                                 | Cardinality  | Example URI                                          | v1.4.0 |
| -------------------- | ------------------------------------------- | ------------ | ---------------------------------------------------- | ------ |
| `cpsv:PublicService` | A service provided by a public organization | Required (1) | `<https://regels.overheid.nl/services/aow-leeftijd>` | ‚úÖ     |
| `cpsv:Rule`          | üÜï A rule governing a service               | 0..n         | `<https://regels.overheid.nl/rules/rule-001>`        | ‚úÖ     |

#### Properties

| Property        | Domain        | Range     | Cardinality | Description                       | Example       | v1.4.0 |
| --------------- | ------------- | --------- | ----------- | --------------------------------- | ------------- | ------ |
| `cpsv:produces` | PublicService | cv:Output | 0..n        | Describes service outputs/results | Output entity | ‚úÖ     |

**Note:** `cpsv:follows` has been **deprecated** in CPSV-AP 3.2.0. Use `cv:hasLegalResource` instead.

### DCT (Dublin Core Terms) (`dct:`)

**Namespace URI:** `http://purl.org/dc/terms/`  
**Standard:** Dublin Core Metadata Initiative  
**Purpose:** Standard metadata properties for titles, descriptions, dates, and identifiers.

#### Properties

| Property          | Domain        | Range   | Cardinality | Description                         | Example                                                            | v1.4.0 |
| ----------------- | ------------- | ------- | ----------- | ----------------------------------- | ------------------------------------------------------------------ | ------ |
| `dct:identifier`  | Any           | Literal | 1..n        | üÜï **Explicit identifier**          | `"aow-leeftijd"`                                                   | ‚úÖ     |
| `dct:title`       | Any           | Literal | 1..n        | Primary name/title                  | `"AOW Leeftijdsbepaling"@nl`                                       | ‚úÖ     |
| `dct:description` | Any           | Literal | 0..n        | Detailed description                | `"Berekening van de AOW-leeftijd..."@nl`                           | ‚úÖ     |
| `dct:language`    | PublicService | URI     | 0..n        | üÜï Language as LinguisticSystem URI | `<https://publications.europa.eu/resource/authority/language/NLD>` | ‚úÖ     |
| `dct:type`        | Output        | URI     | 0..1        | Classification/type                 | `<http://example.org/output-types/certificate>`                    | ‚úÖ     |

**Important Changes (v1.4.0):**

- `dct:identifier` now **explicitly output** for all entities (Service, Organization, Legal Resource, Rule)
- `dct:language` now outputs **URI format** instead of plain string
- `dct:title` now **mandatory for Rules** in addition to other entities

**Note (v1.4.0):** Rules now use **dual typing**: `a cpsv:Rule, ronl:TemporalRule` for CPSV-AP compliance while maintaining Dutch extensions.

## CPSV-AP 3.2.0 Compliance

### Phase 1 Implementation (v1.4.0) ‚úÖ

The Core Public Service Editor v1.4.0 achieves **minimal CPSV-AP 3.2.0 compliance** with the following changes:

#### ‚úÖ Class Type Changes

| Before (v1.3.0)     | After (v1.4.0)                 | Impact                                  |
| ------------------- | ------------------------------ | --------------------------------------- |
| `org:Organization`  | `cv:PublicOrganisation`        | CPSV-AP 3.2.0 compliant class           |
| `ronl:TemporalRule` | `cpsv:Rule, ronl:TemporalRule` | Dual typing for compliance + extensions |

#### ‚úÖ Relationship Changes

| Before (v1.3.0) | After (v1.4.0)        | Impact                           |
| --------------- | --------------------- | -------------------------------- |
| `cpsv:follows`  | `cv:hasLegalResource` | CPSV-AP 3.2.0 compliant property |

#### ‚úÖ New Mandatory Fields

| Entity           | Property         | Cardinality | Status                       |
| ---------------- | ---------------- | ----------- | ---------------------------- |
| **All entities** | `dct:identifier` | 1..n        | ‚úÖ Now explicitly output     |
| **Organization** | `cv:spatial`     | 1..\*       | ‚úÖ **MANDATORY** field added |
| **Rules**        | `dct:identifier` | 1..\*       | ‚úÖ **MANDATORY** field added |
| **Rules**        | `dct:title`      | 1..\*       | ‚úÖ **MANDATORY** field added |

#### ‚úÖ Data Format Changes

| Property       | Before (v1.3.0)       | After (v1.4.0)                                                     | Format                    |
| -------------- | --------------------- | ------------------------------------------------------------------ | ------------------------- |
| `dct:language` | `"nl"`                | `<https://publications.europa.eu/resource/authority/language/NLD>` | LinguisticSystem URI      |

## Usage Patterns

### Complete Service Definition (v1.4.0)

```turtle
@prefix cpsv: <http://purl.org/vocab/cpsv#> .
@prefix cv: <http://data.europa.eu/m8g/> .
@prefix dct: <http://purl.org/dc/terms/> .
@prefix dcat: <http://www.w3.org/ns/dcat#> .
@prefix eli: <http://data.europa.eu/eli/ontology#> .
@prefix foaf: <http://xmlns.com/foaf/0.1/> .
@prefix skos: <http://www.w3.org/2004/02/skos/core#> .
@prefix xsd: <http://www.w3.org/2001/XMLSchema#> .

# Public Service
<https://regels.overheid.nl/services/aow-leeftijd> a cpsv:PublicService ;
    dct:identifier "aow-leeftijd" ;
    dct:title "AOW Leeftijdsbepaling"@nl ;
    dct:description "Berekening van de AOW-leeftijd op basis van geboortedatum"@nl ;
    dct:language <https://publications.europa.eu/resource/authority/language/NLD> ;
    cv:thematicArea <https://standaarden.overheid.nl/owms/terms/aow-leeftijd> ;
    cv:sector <https://publications.europa.eu/resource/authority/corporate-body-classification/NATIONAL> ;
    dcat:keyword "AOW"@nl, "pensioen"@nl, "ouderdom"@nl ;
    cv:hasCompetentAuthority <https://regels.overheid.nl/organizations/svb> ;
    cv:hasLegalResource <https://wetten.overheid.nl/BWBR0002221> ;
    cv:hasCost <https://regels.overheid.nl/costs/aow-leeftijd-1> ;
    cpsv:produces <https://regels.overheid.nl/outputs/aow-leeftijd-1> .

# Organization (CPSV-AP 3.2.0 compliant)
<https://regels.overheid.nl/organizations/svb> a cv:PublicOrganisation ;
    dct:identifier "svb" ;
    skos:prefLabel "Sociale Verzekeringsbank"@nl ;
    foaf:homepage <https://www.svb.nl> ;
    cv:spatial <https://publications.europa.eu/resource/authority/country/NLD> .

# Legal Resource
<https://wetten.overheid.nl/BWBR0002221> a eli:LegalResource ;
    dct:identifier "BWBR0002221" ;
    dct:title "Algemene Ouderdomswet"@nl ;
    dct:description "Wet van 31 mei 1956 tot vaststelling van een algemene ouderdomsverzekering"@nl .

# Rule (dual typing for CPSV-AP + RONL)
<https://regels.overheid.nl/rules/aow-age-2024> a cpsv:Rule, ronl:TemporalRule ;
    dct:identifier "aow-age-2024" ;
    dct:title "AOW-leeftijd bepaling 2024"@nl ;
    dct:description "Regel voor berekening AOW-leeftijd voor personen geboren in 1958"@nl ;
    ronl:validFrom "2024-01-01"^^xsd:date ;
    ronl:validUntil "2024-12-31"^^xsd:date ;
    ronl:confidenceLevel "high" .

# Cost
<https://regels.overheid.nl/costs/aow-leeftijd-1> a cv:Cost ;
    dct:identifier "aow-leeftijd-1" ;
    cv:value "0.00" ;
    cv:currency "EUR" ;
    dct:description "Geen kosten verbonden aan deze dienst"@nl .

# Output
<https://regels.overheid.nl/outputs/aow-leeftijd-1> a cv:Output ;
    dct:identifier "aow-leeftijd-1" ;
    dct:title "AOW-leeftijd"@nl ;
    dct:description "De berekende AOW-leeftijd van de betrokkene"@nl ;
    dct:type <http://example.org/output-types/calculated-age> .
```
